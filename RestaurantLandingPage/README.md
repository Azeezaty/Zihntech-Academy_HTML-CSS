# Restaurant Landing Page

## Project Overview
This project is a responsive landing page for a functional restaurant or cafe. It will showcase the restaurant's brand, featured menu, business hours, location, and photo gallery.

## Planned Features
- Hero section
- About section
- Featured menu
- Business hours
- Photo gallery
- Contact information
- Responsive design

## Technologies
- HTML5
- CSS3

## Status
 This project is currently a placeholder and will be developed as part of the Zhintech Frontend Web Development course.
