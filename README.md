## 👋 Welcome to My Project

This repository contains my **Zihntech Academy Landing Page**, a project I built as part of my frontend development journey.

But for me, this project is more than just a webpage.

It represents a decision I made to take my learning seriously, go back to the fundamentals, and build a stronger foundation in frontend development.

I had previously learned frontend development independently and worked on different projects along the way. However, I realized that I wanted to understand things more deeply and develop my skills through a more structured learning process.

That led me to **Zihntech Academy**.

This project is one of the steps in that journey.
