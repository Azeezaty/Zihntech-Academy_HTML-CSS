# Zhintech Academy HTML-CSS Projects

## Project Overview

This repository contains my HTML and CSS projects completed as part of the Zhintech Academy Frontend Web Development program. It includes my personal portfolio project and placeholder folders for upcoming landing page projects that will be built throughout the course.

## Repository Structure

### MyPortfolio
My personal portfolio website showcasing my work and progress as a Frontend Web Development student at Zhintech Academy.

**Contents:**
- `assets/` – Contains images and video files used in the project.
- `index.html` – The main webpage.
- `style.css` – Styles for the website.
- `script.js` – JavaScript functionality.

###  RestaurantLandingPage
A landing page for a fictional restaurant or café featuring a menu, business hours, location, and a photo gallery.

###  EventLandingPage
A landing page for a tech meetup or conference featuring an agenda, speakers, and a registration button.

###  ProductLandingPage
A landing page for a single product or app featuring a hero section, product features, and a call-to-action.
